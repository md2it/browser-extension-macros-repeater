## ОКНО

- На данный момент одно окно
- Высота окна зависит от количества и размера элементов. Минимальная высота 10см. Ширина фиксированная 10 см
- Окно открывается при запуске как popup под панелью браузера
- Закрывается:
   - При клике вне popup
   - На время любого из режимов (создания нового, исполнения имеющегося)
- Содержит:
   - Вверху кнопка "NEW macros"
   - Ниже "Дефолтный macros"
   - Ниже список имеющихся macros плашками друг над другом

### Каждый macros
- В строку inline
- Выравнены слева:
   - lucide play (запуск режима исполнения)
   - название (некликабельный текст)
- Выравнены справа:
   - lucide square-pen (открывает модальное окно редактирования)
   - trash-2 (открывает модальное окно удаления)

### Mодальное окно редактирования
- Поле названия
- Поле числа повторов (задаёт число повторов полного цикла кликов при каждом исполнении)
- Список шагов (можно поменять местами или удалить шаги)
- Кнопка сохранить и кнопка отмены (выход из окна только по этим кнопкам)

### Модальное окно удаления
- Предупреждает "Удалить <название>?" красный текст
- "Да" (красный), "Нет" (зелёный) (выход из окна только по этим кнопкам)

### Дефолтный macros
- Только название. Сам macros всё ещё в общем списке и редактируется там же. Но в общем списке он слегка выделен жирным начертанием
- Рядом с полем дефолтного macros есть доп кнопка square-pen (не в списке macros)
- square-pen позволяет выбрать другой macros как дефолтный. При этом происходит замена дефолта
