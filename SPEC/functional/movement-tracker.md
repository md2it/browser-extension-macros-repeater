# ОТСЛЕЖИВАНИЕ ДЕЙСТВИЙ

## НАСТРОЙКА
- По умолчанию включено
- Если выключено, то в DOM не внедряемся, все клики и движения виртуальные
- Если включено, то это единственное исключение, из запрета внедрения в DOM

## ЛОГИКА
- Минималистичный простой overlay, lucide mouse-pointer, #f00, 24х24px
- Отражает виртуальное движение курсора
- При клике на 50мс меняет стиль (mouse-pointer-click, #f00, 36х36) и становится дефолтным
